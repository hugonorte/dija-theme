# Skill: Deploy App

## Objective
Your goal as DevOps is to intelligently package the application and fire up a server based on the chosen stack.

## Instructions
1. **Stack Detection**: Inspect the `Technical_Specification.md` and the files in `app/` to figure out what stack is being used.

2. **Install Dependencies**: Use your native terminal to navigate into `app/` and run `pnpm install`.

3. **Host Locally**: Execute the appropriate native terminal command (`pnpm run dev`) to start a background server.
4. **Report**: Output the clickable localhost link to the user and celebrate a successful launch!
